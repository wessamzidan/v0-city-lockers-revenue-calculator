## Overview

This revenue calculator helps sales professionals quickly generate partnership proposals for CityLockers smart locker installations. It features real-time financial projections, scenario comparison, and PDF proposal generation.

**Live Demo:** [v0-city-lockers-revenue-calculator.vercel.app](https://v0-city-lockers-revenue-calculator.vercel.app)

## Features

| Feature | Description |
|---------|-------------|
| **Dashboard** | Real-time annual income projections, revenue mix charts, space utilization gauge |
| **Configuration** | Full business setup with locker specs, pricing, property metrics |
| **Scenarios** | Save, load, compare, and export multiple configurations |
| **Proposal** | Print-ready A4 layout with executive summary and 3-year outlook |
| **AI Assistant** | Trained chatbot for product Q&A with CityLockers knowledge base |
| **WhatsApp CTA** | Floating button for instant partner communication |

## Quick Start

\`\`\`bash
# Clone the repository
git clone https://github.com/wessamzidan/v0-city-lockers-revenue-calculator.git
cd v0-city-lockers-revenue-calculator

# Install dependencies
npm install

# Set environment variables (optional - for AI assistant)
echo "OPENROUTER_API_KEY=your_key_here" > .env.local

# Run development server
npm run dev
\`\`\`
