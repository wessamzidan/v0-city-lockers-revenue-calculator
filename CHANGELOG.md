# Changelog

All notable changes to the CityLockers Revenue Calculator.

## [1.0.0] - 2025-11-25

### Added
- **Dashboard Page**
  - Real-time annual income projections
  - Revenue share inline editor (+/- buttons)
  - "Zero Capex - Zero Opex" business model badge
  - Revenue mix visualization (Luggage, Scooters, Transfers)
  - Space utilization gauge
  - Value proposition cards

- **Configuration Page**
  - Client information inputs (name, location, property type)
  - Contract term selector
  - Locker inventory management with specifications
  - Detailed locker dimensions and "what fits" guidance
  - Scooter service toggle and configuration
  - Transfer service with frequency options
  - Property metrics (keys/units, daily traffic)
  - Pricing reference tables
  - Contextual help tips

- **Scenarios Page**
  - Save configurations with custom names
  - Load saved scenarios
  - Delete scenarios
  - Export as JSON file
  - Import from JSON file
  - Visual scenario cards with key metrics

- **Proposal Page**
  - Print-optimized A4 layout
  - Executive summary section
  - 3-year revenue outlook table
  - Scope of work details
  - Professional contact footer

- **AI Assistant**
  - Floating chatbot interface
  - Trained on CityLockers knowledge base
  - Calculator context awareness
  - Formatted markdown responses
  - Website redirect for official inquiries

- **Navigation & UX**
  - Responsive header with navigation
  - Footer with contact info and social links
  - Floating WhatsApp CTA button
  - Mobile-first responsive design
  - Disclaimer notice

- **Technical**
  - Next.js 15 App Router
  - TypeScript throughout
  - React Context state management
  - localStorage persistence
  - Server Actions for AI
  - Vercel deployment

### Security
- API keys protected via Server Actions
- No sensitive data exposed to client
